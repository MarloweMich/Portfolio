Planet Planters is a fun, retro forum site for plant enthusiasts. Interest in foraging and gardening on the whole has been on the uptick in recent years. To match the influx of learners, Planet Planters has been created. It is designed to be an easy and interactive way to post about your local plantlife, as well as see and comment on posts from other users! It is not location based, so any user may see a post from potentially anywhere if they click into the plant file itself.

Tech Blog is a useful and straight-to-the-point forum site for people interested in technology and sharing tips and tricks with their tools and techs. A homepage feed holds all entries added by all users, each with a link to their own page where discussions through comments are created by all interested users. All entries can be viewed by anyone entering the site, however, ability to post and comment is relegated to those who have logged in. After logging in the user has a dashboard page from whence they can post submissions as well as delete any submissions that they have made previously.

Team Profile Generator is a command-line application for inputting employee's into three categories: manager, engineer and intern. Each employee is presented with an automatically generated ID as well as contact information for ease of access. It's a primarily backend application using node.js and the inquirer package to create an HTML file after it runs. There is no deployed site but there is a youtube video demonstrating it's functionality with a heavy emphasis on Andrea Bocelli

BougieFoodie is a project enabling the user to stand at the intersection of food and culture, having a central hub for the best user-generated content on any recipe or food item by leveraging the API's of both YouTube and Wikipedia. It's a great tool to learn how to make a dish and be informed about it's history and cultural significance.

Weather Dashboard is a useful tool for looking up the weather anywhere in the world and getting an accurate picture of the current day's conditions as well as the five days afterward. The site site makes use of the OpenWeatherApi and Moment.js in order to keep an accurate record of the current time and weather. A useful website for planning a trip to the park or for those curious to find out just how rainy it is in London.
This site is live Right Here

Workday Scheduler is a simple yet effective space to plan out the day with whatever events and todos one will inevitably find themselves needing to remember. It uses Moment.js to color code all hours in a typical workday to indicate whether they are past, currently happening, or yet to be addressed. Just fill out the schedule with your morning coffee. Preparation is the key to success.
